Jasmineについて
    JasmineはOS/2 Warpで動作する、ホームページ用ファイルのアップロード用ユーティリティ・ソフトです。 FTPアップロード機能の他に、ファイルのコピー機能も備えています。 Jasmineは変更のあったファイルを自動的に判別し、指定されたサーバー、ディレクトリにファイルをアップロード、コピーします。 また、アップロード、コピーしたファイルを管理し、相手のサーバー、ディレクトリからアップロード、コピーしたファイルを削除する機能も備えています。

    JasmineはVX-REXXで作成されていますので、実行にはVX-REXXのRuntime Library `VROBJ.DLL'が必要となります。 `VROBJ.DLL'は下記より入手してください。

    • Download VROBJ.DLL 2.1d (vrobj21d.zip)

インストール
  • 古典的インストール方法

    1. jsm???r.zipを適当なディレクトリにコピーし、解凍します。

    2. install.cmdを実行するとJasmineのオブジェクトがデスクトップに作成されます。

  • WarpINでインストールする (WarpINがインストールされている必要があります)

    1. jsm???r.wpiのアイコンをダブル・クリックする。

    2. WarpINの指示に従い、インストールをする。

    WarpINは下記ホームページから入手出来ます。

    • xworkplace.org: WarpIN

使用方法
  • サイトの登録と削除

    • FTPサイトの登録

      1. `New'ボタンを押し、FTPサイトを登録してください。
      2. Modeは`FTP'を選択してください(初期値)。
      3. 適宜、Address, User ID, Password, Account, Local Dir, Remote Dir, Exit Commandを入力してください。

        例：朝日ネットの場合

          Address :netftp.asahi-net.or.jp
          User ID :(your user id)
          Password :(your password)
          Account :(blank)
          Local Dir :f:\wwwhome\index
          Remote Dir :public_html
          Exit Command:(optional command)

      4. 入力が終了したら`Save'ボタンを押して設定をSaveしてください。

    • COPYサイトの登録

      1. `New'ボタンを押し、COPYサイトを登録してください。

      2. Modeは`COPY'を選択してください。

      3. `Source'にコピー元のディレクトリを、`Destination'にコピー先のディレクトリを入力してください。

      4. 入力が終了したら`Save'ボタンを押して設定を保管してください。

    • サイトの削除

      • 削除するサイトを選択し、`Delete'ボタンを押すと選択されたサイトが削除されます。

  • 実行方法

    • FTPの場合

      1. インターネット接続がされていることを確認してください。

      2. `Run'ボタンを押すとアップロードが開始されます。

    • COPYの場合

      1. Source/Destinationにリムーバブル・ディスクやネットワーク・ドライブ等を指定している場合はドライブが接続されていることを確認してください。
      2. `Run'ボタンを押すとコピーが開始されます。

  • 機能

    • ログの保存

      • ログを保管するには、`Log'チェック・ボックスをチェックしてください。

      • ログ・ファイルは日毎、月毎にまとめて保存することができます。

      • ログの詳細を専用のログ・ビューワーで確認することができます。

      • ログ・ビューワーの`Clear All Log Files'ボタンを押すとログ・ファイルをすべて消去することができます。

    • アップロード、コピーの機能

      1. ログオンの再実行(FTPモード)
        ログオンに失敗した時に再度ログオンを行いたい場合、`Logon Retry'チェック・ボックスをチェックしてくささい。 また、再実行の回数も設定出来ます。

      2. 大文字を小文字に変換する機能
        アップロード、コピー時にファイル名の大文字を小文字に変換する必要がある場合、`Change UPPER to lower'チェック・ボックスをチェックしてください。

      3. アップロード、コピーするファイルの選択
        アップロード、コピーするファイルの一覧を表示するには、`Upload File Check'チェック・ボックスをチェックします。 `Run'ボタンを押した後、`Upload File Check'ウィンドウが開きます。 ここで、アップロード、コピーしないファイルを選択することが出来ます。

      4. アップロード、コピーのステイタスを記憶する
        上記、`Upload File Check'ウィンドウで無効にしたファイルを次回以降も無効にすることができます。 この機能を使用する場合は`Remember Upload Status'チェック・ボックスをチェックしてください。

        【注意】 この機能は`Upload File Check'機能が有効の時のみ使用することができます。

      5. アップロードを失敗したファイルを次回再度アップロードする
        何等かの原因(電話回線品質の問題等)でアップロードに失敗したファイルを次回アップロード時に再度アップロードすることができます。 この機能を使用する場合は`Retry Failed File'チェック・ボックスをチェックしてください。

      6. ファイル拡張子でアップロード、コピーしないファイルの設定
        アップロード、コピーしないファイルの拡張子を指定することができます。
        この機能を使用する場合は、`Ignore File Extension'チェック・ボックスをチェックしてください。 また、必要な拡張子を追加してください。

      7. 必ずアップロードするファイルの設定(都度設定)
        前回のアップロード時から変更されておらず、今回のアップロード対象とならないファイルをアップロードする場合、 `Run'ボタンを押す前に、`Run After Compulsory Upload Files'チェック・ボックスをチェックしてください。

        【注意】 この機能は`Sequential Run Mode'が有効時には`Selected Site Only'モードでのみ利用可能です。

      8. 常にアップロードするファイルの指定
        メイン・ウィンドウに表示されている`Set Always Upload Files'ボタンを押すと、`Set Always Upload Files'のウィンドウが開きます。 ここで、常にアップロードするファイルを指定する事ができます。 また、アップロードする感覚を1時間毎に設定出来ます。

      9. アップロード、コピー対象外ファイルの指定
        メイン・ウィンドウに表示されている`Set Excluded Files'ボタンを押すと、`Set Excluded Files'のウィンドウが開きます。 ここで、アップロードまたはコピーの対象としないファイル、ディレクトリを指定できます。

    • ファイル転送モードの選択 (FTPモード)

      • ファイル転送するモードをASCII/Binary/Autoから選択します。

      • ファイル転送モードを`Auto'に設定すると、ASCII Listに登録された拡張子を持つファイルはASCII Modeで、 それ以外のファイルはBinaryモードでアップロードされます。

    • アップロード・ファイルのステイタスの表示

      • ファイル毎の転送モードの表示(FTPモード)
        アップロードしたファイル毎の転送モード(ASCII/Binary)を表示することができます。 この機能を使用する場合は、`Transfer Mode'チェック・ボックスをチェックしてください。

      • ファイル毎のサイズの表示
        アップロードしたファイル毎のサイズを表示することができます。 この機能を使用する場合は、`File Size'チェック・ボックスをチェックしてください。

      • アップロード、コピーの結果の表示
        アップロード、コピーの結果を表示することができます。 この機能を使用する場合は、`Upload Summary'チェック・ボックスをチェックしてください。

    • 登録サイトを連続して実行する

      • 登録されたサイトを連続してアップロード、コピーすることができます。

      • 連続実行中はサウンド機能を無効にする場合、`Disable Sound During Sequential Running'チェック・ボックスをチェックしてください。

    • 実行中のアップロード、コピーのキャンセル

      • アップロード、コピーの実行中にキャンセルするには、`Cancel'ボタンを押してください。
        `Cancel'ボタンは`Run'ボタンを押すと表示されます。

    • アップロード、コピー終了後に外部のプログラムを実行する

      • アップロード、コピー終了後に外部のプログラムを実行する場合に設定します。
        実行するプログラム名は各サイト毎に設定します。

    • サウンド機能

      • 各イベント終了毎に任意のWAVファイルを鳴らす事ができます。
        各サウンドの割り当ては`Setup'ボタンを押して設定してください。

    • マルチ・リンガル化

      • 画面に表示されるテキストを日本語(または他国語)に変更することができます。 詳細はサンプルのLanguage File, english.lng を参照してください。 また、いつでも初期値に戻すことができます(Jasmineの再起動が必要)。
        【注意】 Language FileはJasmineと同じディレクトリになければいけません。 また、パラメータの追加・削除はしないでください。

      • 添付されているLanguage File

        1. english.lng : 英語版 (Sample)
        2. japanese.lng : 日本語版
        3. deutsch.lng : ドイツ語版
        4. portuguese.lng : ポルトガル語版
        5. french.lng : フランス語版

  • HTMLキーワードの変換 (FTPモード)

    • アップロード時にHTMLファイルに記述されている%で囲まれたキーワード(下記表参照)を実データに変換する場合、`HTML Keyword Exchange'チェック・ボックスをチェックしてください。

        アップロードした年 : %JSM_UPLOAD_YEAR%
        アップロードした日 : %JSM_UPLOAD_DATE%
        アップロードした時間 : %JSM_UPLOAD_TIME%
        ローカル・ファイルの最終更新年 : %JSM_UPDATE_YEAR%
        ローカル・ファイルの最終更新日 : %JSM_UPDATE_DATE%
        ローカル・ファイルの最終更新時間 : %JSM_UPDATE_TIME%
        オリジナル・ファイルのサイズ : %JSM_FILE_SIZE%
        指定したファイルの最終変更日 : %JSM_UPDATE_DATE=file_name%
        指定したファイルの最終変更時間 : %JSM_UPDATE_TIME=file_name%
        指定したファイルのサイズ : %JSM_FILE_SIZE=file_name%
        file_name = 絶対パスまたは相対パスで指定したファイル名

        指定した複数ファイルの最終
        更新日/時間を表示         		: %JSM_LATEST=fn_1, fn_2, ..., fn_n:TYPE%
        fn_x = 絶対パスまたは相対パスで指定したファイル名(複数指定可)
        TYPE = DATE, TIME, DATETIME, ALL, BOTH (DATETYPE=ALL=BOTH=null)

        上記キーワードの`%'の前後にコメント・タグ(`<!--'と`-->')を付記することが出来ます。 これにより、ローカル・ファイルでホームページを表示する際、長いキーワードの代わりに`%%'が表示される様になり、デザインを壊すこと無くホームページを表示出来るようになります。
        コメント・タグの付記方法は下記の例を参照にしてください。

          通常のキーワード コメント・タグを付記したキーワード
          %JSM_UPLOAD_DATE% %<!--JSM_UPLOAD_DATE-->%
          %JSM_UPDATE_DATE=file_name% %<!--JSM_UPDATE_DATE=file_name-->%

    • `Date Separator'パラメータ
      日付区切り文字をSlash(/)またはDot(.)のいずれかに設定出来ます。

    • `Month Indication'パラメータ
      月の表記方法を数字(01,02...12)またはアルファベット(Jan,Feb...Dec)のいずれかに設定出来ます。

  • `Takeout'機能

    • Takeout機能を使用してJasmineがアップロード、コピーしたファイルを削除することができます。

    • Takeout機能を使用するにはまずPreferenceの`Takeout'チェック・ボックスをチェックし、Takeout機能を有効にしてください。 Tankeout機能が有効になると`Takeout'ボタンがメイン・ウィンドウに表示されます。

    • Takeout機能が有効の場合、Takeout対象ファイルはアップロード、コピーが行われる毎にアップデートされます。

    • 最初のTakeout実行時にログ・ファイルを検索し、Takeoutデータ・ファイルを作成します。 ログ･ファイルの容量によってはしばらく時間がかかる場合があります。

    • 使用方法

      1. `Takeout'ボタンを押すと`Jasmine Takeout'ウィンドウが表示されます。

      2. `Jasmine Takeout'ウィンドウの上部リスト・ボックスにTakeout対象ファイルの一覧が表示されます(下部リスト･ボックスにはTakeoutの状況が表示されます)。

      3. Takeoutするファイルを選択します(複数選択する場合はCtrlキーを押しながら選択してください)。

      4. `Takeout!'ボタンを押すとログオンが開始され(FTPモードの時)、順次ファイルの削除を行って行きます。

      5. 削除されたファイルはTakeout対象ファイル一覧からも削除されます。

      NOTE:

      • 相手側にないファイルを削除しようとした場合、ステイタス表示にはFailと表示されます。

    • その他のTakeout機能

      1. Takeout対象ファイルから任意のファイルを削除。

        1. Takeout対象ファイルから外したいファイルを選択します(複数選択する場合はCtrlキーを押しながら選択してください)。

        2. `Clear Takeout Data'ボタンを押すと、Takeout対象ファイルから選択されたファイルが削除されます。

      2. Takeout対象ファイルから現在ローカル・ディレクトリにないファイルだけを選択する。

        • `Select Files NOT Available at Local/Source Directory'ボタンを押すと検索を始めます。

        • 検索が終了するとすでにローカル・ディレクトリにないファイルだけが選択された状態で表示されます。

      3. Takeout対象ファイルを再構築する。

        • `Create Takeout Data'ボタンを押してください。 再度ログ・ファイルの検索を行い、Takeout対象ファイルを再構築を行います。

  • タイマー機能

    • 実行モード

      1. Run Regularly
        Jasmine起動時から、指定された経過時間毎に設定したサイトのアップロード、コピーを行います。

      2. Run Fixed Time
        指定した時間に設定したサイトのアップロード、コピーを行います。

    • Timer Runの機能

      • OS/2の起動時にJasmineを自動で実行させる
        OS/2の始動フォルダにJasmineのオブジェクトを作成することが出来ます。 これにより、OS/2起動時にJasmineを自動で実行させることが出来ます。 JasmineのオブジェクトをOS/2の始動フォルダに作成するには、`Create Object at Startup Folder'チェック・ボックスをチェックしてください。

      • 最小化で実行 Minimized Startup
        Jasmineを起動時に最小化するには、`Minimized Startup'チェック・ボックスをチェックしてください。

      • 経過時間を表示する
        Jasmine起動時からの経過時間を表示するには、`Show Timer Run'チェック・ボックスをチェックしてください。 経過時間はメイン・ウィンドウに表示されます。

      • Jasmine起動時にサイトのアップロード、コピーを実行する。(`Run Regularly'モードのみ)
        Jasmine起動時にサイトのアップロード、コピーを実行するには`Run at Startup' check boxをチェックしてください。 この機能は、`Run Regularly'モードのみ有効です。
