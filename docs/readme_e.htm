Jasmine
    Jasmine is the ftp front-end using of the OS/2 WARP genuine part for FTP (File Transfer Protocol) for the maintainances of your webpages. And Jasmine can copy local files to destination directory with same functions of FTP Mode. Jasmine will put/copy the files and directories, only which are changed or altered from the last put/copy. You can also select if you want to put/copy this time such automatically selected files and folders. You may record the logs of uploaded files by either daily or monthly, too. And you can remove remote files that uploaded/copied and managed by Jasmine.

    This OS/2 application requires VROBJ.DLL, the runtime module from Watcom VX-REXX.
    You can get a copy from...

    • Download VROBJ.DLL 2.1d (vrobj21d.zip)

Install
  • Legacy way

    1. Unzip the archived file (jsm???r.zip) into any unique local directory.

    2. Run INSTALL.CMD by either OS/2 full screen or OS/2 session. The Jasmine program object will automatically be created on your desktop.

  • Install using WarpIN (WarpIN should be installed)

    1. Run jsm???r.wpi file and go ahead.

    You can get WarpIN from...

    • xworkplace.org: WarpIN

Usage
  • Create/Delete Site

    1. Push the `New' button to create the site.

      1. Register FTP Site
        Select `FTP' radio button (default), and type correctly each and every information of your FTP site as: Address, User ID, Password, Account, Local Dir, Remote Dir. For Instance:

          Address :ftp.jasmine.com
          User ID :(your user id)
          Password :(your password)
          Account :(blank)
          Local Dir :f:\wwwhome\index
          Remote Dir :public_html
          Exit Command:(optional command)

        • Above are the case if your FTP site is `ftp.jasmine.com'.
        • Your local directory is `f:\wwwhome\index'.
        • Remote directory of your FTP site is `homepage'.

      2. Register COPY Site
        Select `COPY' radio button, and type correctly each and every information of your COPY site as: Source and Destination. For Instance:

          Source :f:\wwwhome\index
          Destination:g:\backup

        • Above are the case if your source directory is `f:\wwwhome\index'.
        • Your destination directory is `g:\backup'.

      Please push the `Save' button after you input.

    2. Push the `Delete' button to delete the site
      You may delete the saved FTP/COPY site information by pushing the `Delete' button.

  • Basic Usage

    1. FTP Mode

      1. Dialup to your Host Provider if you are contracted ISV(InternetServiceProvider) based with Dialup User Service. Otherwise, check that you are now connected to the Internet so far.

      2. Check the contents in the indication, and click the `RUN' button when everything is O.K.

      3. The program will login to your FTP site, and start to transfer the contents which are made and/or altered after the date and the time of `the Last Update'.

      4. Watch the status window and click the `EXIT' when it's done!

        Attention
        Don't forget log off the dialer especially if you are the Dialup user!!

    2. COPY Mode

      1. Click the `RUN' button when everything is O.K.

      2. The program will start to copy the contents which are made and/or altered after the date and the time of `the Last Update'.

      3. Watch the status window and click the `EXIT' when it's done!

      Attention
      Please note that the program will automatically put/copy the files and folders, which means that the current files on the FTP server/destination directory under the same name will be over written and you may not be able to hold them anyway!

  • Functions

    1. Logging

      • Check `Logging' check box, if you want to log status.

      • The daily or monthly, as you selected, log files will be kept in the log folder under the `Jasmine' folder.

      • Click `View Log File' button if you want to see log details.

      • Click `Clear All Log Files' button on Log Viewer if you want to delete all log files.

    2. `Before running, select compulsory upload files' Function
      Check `Before running, select compulsory upload files' check box if you want to upload/copy NOT UPDATED files.

      Attention
      When `Sequential Run Mode' is enabled, this function is valid at `Selected Site Only' mode.

    3. `Set Always Upload Files' Function
      Click `Set Always Upload Files' button if you want to set files which you want to upload evertime when Jasmine run. You can set interval time, too.

    4. `Set Excluded Files' Function
      Click `Set Excluded Files' button if you want to set files/directories which you don't want to upload/copy.

    5. Upload/Copy Functions

      1. `Logon Retry' Function (FTP Mode)
        Check `Logon Retry' check box if you want retry logon when logon is failed. You may select retry number.

      2. `Change UPPER to lower' Function
        Check `Change UPPER to lower' check box if you want change all UPPER case to lower case when Jasmine will upload/copy.

      3. `Manage Upload Files' Function

        • `Upload File Check' Function
          You may check the automatically seleceted files and folders once the `Upload File Check' check box is checked otherwise program will put/copy all the files and folders being automatically selected by the program, which might have the newer time stamp.

        • `Remember Status' Function
          Check `Remember Status' check box, if you want Jasmine to remember the status you set at `Upload File List' window.

      4. `Remember Upload Status' Function
        Check `Remember Upload Status' check box if you want to remember files that you set `Disabled' at `Upload File Check' window.
        Attention
        This function is valid when `Upload File Check' function is enabled.

      5. `Retry Failed File at Next Time' Function
        Check `Retry Failed File' check box if you want to retry upload files that was failed at previously.

      6. `Ignore File Extension' Function
        Check `Ignore File Extension' check box if you doesn't want to upload/copy files that has same extension.

    6. File Transfer Mode (FTP Mode)

      • You may select file transfer mode from ASCII/Binary/Auto.

      • When you select `Auto', all files will be transfered by Binary mode except files entered at `ASCII Mode' list.

    7. Show Upload Status

      1. Show Transfer Mode (FTP Mode)
        Check `Transfer Mode' check box if you want to indicate transfer mode every file.

      2. Show File Size
        Check `File Size' check box if you want to indicate every upload/copy file size.

      3. Show Upload Summary
        Check `Upload Summary' check box if you want to indicate upload/copy status.

    8. `Sequential Run' Function

      • Check `Sequential Run Mode' check box if you want to use sequential upload/copy function.

      • Check `Disable Sound During Sequential Running' check box is you want to turn off the sound during sequential running.

    9. `Cancel Upload/Copy' Function

      • Click `Cancel' button if you want to cancel upload/copy. This button will show after click `Run' button.

    10. `Exit Command' Function

      • Check `Exit Command' check box if you want to run any program after upload/copy is completed. Exit command should be set by Site.

    11. Sound

      • You may set sound set by clicking `Setup' button.

    12. Language Support

      • You can customize indication of labels and buttons by your language. Parameters, see sample language file, english.lng. And anytime you can reset default labels (Jasmine restart is required).

        Attention
        Language files should exist at same directory of Jasmine.
        * Don't add/delete any parameters!

      • Attached Language Files

        1. english.lng : `English' Language File (Sample)
        2. japanese.lng : `Japanese' Language File
        3. deutsch.lng : `Deutsch/German' Language File
        4. portuguese.lng : `Portuguese' Language File
        5. french.lng : `French' Language File

  • `HTML Keyword Exchange' Function (FTP Mode)

    1. Check `HTML Keyword Exchange' check box if you want to exchange keywords (see below) to real data in HTML files when upload it.

        Exchange to upload year : %JSM_UPLOAD_YEAR%
        Exchange to upload date : %JSM_UPLOAD_DATE%
        Exchange to upload time : %JSM_UPLOAD_TIME%
        Exchange to update year : %JSM_UPDATE_YEAR%
        Exchange to update date : %JSM_UPDATE_DATE%
        Exchange to update time : %JSM_UPDATE_TIME%
        Exchange to original file size : %JSM_FILE_SIZE%
        Last update date of designated file : %JSM_UPDATE_DATE=file_name%
        Last update time of designated file : %JSM_UPDATE_TIME=file_name%
        Designated file size. : %JSM_FILE_SIZE=file_name%
        file_name = file name with full path or relative path.

        Latest Update Date/Time of designated files : %JSM_LATEST=fn_1, fn_2, ..., fn_n:TYPE%
        fn_x = file_names with full path or relative path.
        TYPE = DATE, TIME, DATETIME, ALL, BOTH (DATETYPE=ALL=BOTH=null)

      You can include comment tag (`<!--' and `-->') in above keywords. It means you can become to design your homepage without long keywords. Normaly when you use `%JSM_UPLOAD_DATE%' in your homepage, your web browser just shows `%JSM_UPLOAD_DATE%' (at local page). But when you include comment tag in the keyword, it only shows `%%' instead of long keyword.
      Please refer following sample for how to use this function.

        Normal Style Including Comment Tag
        %JSM_UPLOAD_DATE% %<!--JSM_UPLOAD_DATE-->%
        %JSM_UPDATE_DATE=file_name% %<!--JSM_UPDATE_DATE=file_name-->%

    2. `Date Separator' parameter
      You can select date separator from SLASH(/) and DOT(.).

    3. `Month Indication' parameter
      You can select month indication from Number (01,02...12) and Alphabet (Jan, Feb...Dec).

  • `Takeout' Function

    1. You can remove remote files that uploaded/copied and managed by Jasmine.
      For using Takeout function, at first check `Takeout' check box at Preference. When `Takeout' function become enabled, you will find `Takeout' button at Jasmine main window.

    2. When Takeout function is enabled, takeout data file is updated after every upload.
      Jasmine make Takeout Data File at first startup of Takeout. It might take longer time for searching log file if there are many log files.

    3. How to use

      1. Jasmine Takeout window will available when click `Takeout' button.

      2. You may now look the takeout candidates with its status at the list box `Jasmine Takeout' window.

      3. Select files that you want to takeout with use of Ctrl key.

      4. Click `Takeout!' button, then Jasmine start to remove selected files form host.

      5. All removed files are removed from Takeout Data File.

      NOTE:

      • You might be noticed as `Failed' when you will delete the files which are not available at your host/destination directory.

    4. Other Takeout Functions

      1. Eliminate the files from Takeout candidates.

        1. You may select files which you like to eliminate from the candidates with use of Ctrl key when the files are more than one.

        2. Push `Clear Takeout Data' button.
          The files you like to eliminate from the candidates will be eliminated from the candidates.

      2. Select only files not existing at your local/source directory.
        Jasmine can automatically search files which are not available on local/source directory.

        • Click `Select Files NOT Available at Local/Source Directory' button, then start search files.

        • After search is finished, only NOT Available at Local/Source Directory files are selected.

      3. Reconstruct Takeout Data.

        • Push `Create Takeout Data' button will recover the candidates which you have eliminated from the candidates to the candidates once again by searching log files.

  • Timer Run

    1. Run mode

      1. Run Regularly
        Jasmine run selected site by appointed time intervals from start up.

      2. Run Fixed Time
        Jusmine run selected site at appointed time.

    2. Functions

      1. Create Object at Startup Folder
        Check `Create Object at Startup Folder' check box, if you want to create object of Jasmine in Startup folder.

      2. Minimized Startup
        Check `Minimized Startup' check box, if you want to startup Jasmine as minimized (hidden).

      3. Show Timer Counter
        Check `Show Timer Run' check box, if you want to show Timer Run counter on Jasmine main window.

      4. Run at Startup (`Run Regularly' mode only)
        Check `Run at Startup' check box, if you want to upload at Jasmine startup time.
